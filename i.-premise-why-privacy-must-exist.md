# I. Premise: Why Privacy Must Exist

[English](#english) | [中文](#中文) | [Русский](#русский) | [日本語](#日本語)

![](.gitbook/assets/image)

***

## English

Voidify is a decentralized, non-custodial protocol built on Solana. Users deposit crypto assets and later withdraw them using zero-knowledge proofs, creating unlinkable transfers between wallets without relying on an intermediary.

Solana provides speed and low transaction costs, but every normal transaction is publicly visible. This exposes user activity and weakens financial confidentiality. Voidify addresses that gap through fixed-denomination on-chain privacy pools: a user can deposit into a pool and later withdraw to another wallet without a visible on-chain link.

Voidify combines Solana performance with modern zero-knowledge cryptography. Built natively in Rust, the protocol is designed for self-custodial interaction without giving up decentralization or user control.

## 中文

Voidify 是构建于 Solana 上的去中心化、非托管协议。用户可以存入加密资产，并在之后通过零知识证明提款，从而在不同钱包之间形成无法直接关联的转移，无需依赖中介。

Solana 具备高速和低交易成本优势，但普通交易都公开可见，这会暴露用户行为并削弱金融隐私。Voidify 通过固定面额的链上隐私池解决这一问题：用户可先存款，再将资产提取到另一个钱包，而链上不会留下明显的对应关系。

Voidify 将 Solana 的性能与现代零知识密码学结合。协议以 Rust 原生构建，面向自托管交互设计，同时保持去中心化和用户控制权。

## Русский

Voidify — это децентрализованный некастодиальный протокол на Solana. Пользователи вносят криптоактивы и позже выводят их с помощью доказательств с нулевым разглашением, создавая несвязываемые переводы между кошельками без посредника.

Solana обеспечивает высокую скорость и низкую стоимость транзакций, но обычные транзакции полностью публичны. Это раскрывает действия пользователей и ослабляет финансовую конфиденциальность. Voidify решает проблему с помощью ончейн-пулов фиксированного номинала: пользователь может внести средства и затем вывести их на другой кошелек без видимой связи в блокчейне.

Voidify сочетает производительность Solana с современной криптографией нулевого разглашения. Протокол нативно разработан на Rust для самостоятельного хранения средств без отказа от децентрализации и контроля пользователя.

## 日本語

Voidify は Solana 上に構築された、分散型かつノンカストディアルなプロトコルです。ユーザーは暗号資産を預け入れ、後からゼロ知識証明を用いて出金することで、仲介者に依存せずウォレット間の結び付きを見えなくできます。

Solana は高速で低コストですが、通常のトランザクションはすべて公開され、ユーザーの活動や金融上の機密性が損なわれます。Voidify は固定額のオンチェーン・プライバシープールにより、この問題に対応します。ユーザーはプールへ預け入れ、後から別のウォレットへ、明確なオンチェーン上の関連を残さず出金できます。

Voidify は Solana の性能と現代的なゼロ知識暗号を組み合わせています。Rust でネイティブに構築され、分散性とユーザーの管理権を損なわないセルフカストディ型の利用を目指します。
